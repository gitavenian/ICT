Key Features

Stable Performance Thanks to a Design Optimized for Multicore Systems

  • Optimal Performance:

    The multicore distribution algorithm enhances performance by efficiently utilizing CPU cores, eliminating idle time.

  • Swift Rule Application:

    Employing a Packet Classification Algorithm ensures that the speed of rule application and discovery remains unaffected by the quantity of security policies in place.

Security Policies Focused on IP Address and User ID

  • Enhance both security and user convenience by implementing user policies based on user IDs connected to diverse authentication systems, thereby minimizing the complexity of IP management.
  • Simplify user access management by enabling support for third-party authentication and facilitating user ID/password authentication without constraints on devices.
  • Bolster network security through access control measures such as MAC address authentication and the monitoring of user-specific traffic statistics.

Security Customization at the Unit Level with Zone-Based Policies

  • Zone-based security policies leverage interface groups to assign physical interfaces to pre-established zones, bolstering network identity and security.
  • To optimize efficiency, security modules such as Content Filters, Intrusion Prevention Systems (IPS), Anti-DDoS, and Firewalls are selectively deployed to designated security zones, preventing performance decline and strengthening security with tailored policies.
  • This strategy enables adaptable zone assignments based on internal network configurations, facilitating convenient adjustments for diverse security policy allocations and simplifying the management of policy changes.

dCube Bonding – Providing a stable communication environment with our dCube solution

  • The system binds low-cost Internet lines to create high-bandwidth VPN channels and optimizes line management to ensure uninterrupted services for increased communication demands.
  • Bandwidth-based load balancing maximizes the speed of multiple Internet lines and minimizes lag/jitter caused by packet loss.
  • Flow control mechanisms are implemented to maintain tunneling and prevent excessive data inflow during potential security threats or data overload.

Various Routing Protocols Available

  • Static, RIP, OSPF Routing, and PBR.
  • User-based routing enables the assignment of distinct routes for each user.
  • Optimize network/system resources with Multicast Routing support for PIM-DM, PIM-SM, and IGMP.
